VS插件

Visual Studio F12 进入到 反编译源代码 (推荐)

在工具 > 选项 >文本编辑器 >C# >高级 >分析 , 中有一个 支持导航到反编译源(实验). 将它勾选上.
然后我们在 使用 F12 就可以进入到 反编译出来的源码了.
![在这里插入图片描述](https://img-blog.csdnimg.cn/2019042513094485.jpg =600x550)
结果:
![在这里插入图片描述](https://img-blog.csdnimg.cn/20190425133533247.jpg =700x400)

Visual Studio IntelliCode(推荐)

点击查看详情
..
微软新出的AI 辅助开发.挺好用的
可以根据代码 上下文 智能提示出 你可能需要的代码
图中 标星号的 就是效果
![在这里插入图片描述](https://img-blog.csdnimg.cn/20190425105732731.jpg? =500x300)

IntelliSense Extender (推荐)

查看详情
..
和Resharper 的 智能提示类似.
只要是 引用了dll. 就不再 需要 关注 顶部是否 using 命名空间.
智能提示 后面有命名空间的 就是 当前没有 using . 选择后 自动引用 using.
再也不需要 死记硬背 命名空间了.
![…](https://img-blog.csdnimg.cn/20190425110105179.jpg =500x200)
新版Visual Studio 中也新增了这个功能
在 工具->选项->文本编辑器->C#->IntelliSense 中.
勾选后 也可以实现 自动引用命名空间的效果,
没有使用新版的 小伙伴可以下载上面的这个插件.
![在这里插入图片描述](https://img-blog.csdnimg.cn/2019052717435815.png? =500x300)

ReSharper(推荐,电脑性能要求较高)

耗费性能较多. 不选择这个的小伙伴可以选择 本文中 其他几个 插件. 都可以实现 Resharper 的功能
比如
CodeMaid 代码整理.
IntelliSense Extender 智能提示扩展, 自动using 命名空间
Visual Studio IntelliCode 分析代码,给出需要的提示

ReSharper是一个JetBrains公司出品的著名的代码生成工具，其能帮助Microsoft Visual
Studio成为一个更佳的IDE。你可以进行深度代码分析，智能代码协助，实时错误代码高亮显示，解决方案范围内代码分析，快速代码更正，一步完成代码格式化和清理，业界领先的自动代码重构，高级的集成单元测试方案，和强大的解决方案内导航和搜索。

推荐大佬的网址,学完后 让你成为 骨灰级玩家:
VisualStudio神级插件Resharper技巧基础入门到骨灰玩家使用全教程+Resharper性能优化
练习Resharper各种操作与快捷键的开源项目

CodeMaid(推荐)

代码格式 整理 , 点击 清理活动文档,
可以定位到 解决方案资源管理器中的文件位置, 在项目文件多的情况下 非常好用.

![在这里插入图片描述](https://img-blog.csdnimg.cn/20190403120423163.jpg? =700x400)

OzCode

vs 调试插件, 功能丰富. 调试的时候可以方便的 得到更多的信息

Web Essentials

这是一个前端页面的插件。  可以说是 包含了页面 所有 需要的 功能,

HideMenu

这个插件的唯一功能就是  将上方的菜单栏 隐藏，  当你按下 A l t  再次出现在 上方。 松开它又会隐藏起来。
为窗口腾出更大的空间。  也让整个窗口看起来非常的简洁。
vs 2019 现在 可以将 菜单栏 整合到 顶部了.就不需要这个插件了.

Word Highlight With Margin ( 推荐 )

高亮所选的词 插件,

IndentGuide v15

这个插件的效果呢就是这个样子的，当我们的代码头和尾巴相差太远。 出了屏幕。     或者
多层括号的时候，这个插件就体现出他的价值了。层次非常的清晰，。 新版的vs 已经加入了这个功能. 旧版的没有.需要自己下载插件

ColorfulIDE

背景图 插件
vs的纯色背景是不是看腻了 ！！！
只需要简单的指定一个 文件夹路径。他就可以自动的把图片变成背景。（个人建议 透明度 设置为 0.06 最佳）

SQLServer 插件

SQLPrompt(推荐)

SQLServer的 SQL 关键字 智能提示。比自带的提示好很多. 
唯一的缺点就是  打开SQL Server Management Studio  需要几秒钟初始化,
其他的都是优点
比如:  Tab Color   支持给tab 换颜色.  更加清晰的辨别 测试环境和 生产环境
(根据IP 来设置的颜色.只要是连接上 对应的 数据库  ,tab 就会变色)
![在这里插入图片描述](https://img-blog.csdnimg.cn/20190619150226824.jpg? =700x400)
如何设置:
![>](https://img-blog.csdnimg.cn/20190619150445281.jpg? =700x400)

轻量级 访问SQLServer( 需要VSCode )

在vscode 扩展中 去搜索这个 插件 就可以 连接SQLServer 了.
可以在配置文件中手动去管理 数据库 连接串
功能齐全. 非常轻量级, 打开vscode 就可以 执行SQL

Git 工具

GitKraken(推荐)

  1. 里面的 分支 solo 和 分支查找 功能 都非常的 好用.
  2. 可以在不切换分支的情况下去查看其它分支的代码
  3. 代码对比窗口 也比较大,方便查看代码,支持 直接修改.然后点击保存就可以了.
  4. 还有关键字高亮.
  5. 界面好看, 功能也多.
  6. 至今 我 还没有发现 任何 不方便的 地方.

Sourcetree

之前用的是这个,只有基本的功能,
与 GitKraken 差距较大.
